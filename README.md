\# 🔗 URL Shortener



A full-stack URL shortener with click tracking and analytics, similar to bit.ly.



\## ✨ Features



\- Create short links from long URLs

\- Custom short codes (e.g., `localhost:3000/my-link`)

\- Track every click with timestamp

\- Analytics dashboard with hourly breakdown

\- See traffic sources (where clicks came from)

\- View all your links in one place



\## 🛠️ Tech Stack



| Layer | Technology |

|-------|------------|

| Backend | Node.js, Express.js |

| Database | SQLite |

| Frontend | HTML5, CSS3, JavaScript |
